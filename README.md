# 🎓 Smart Placement Predictor

A Machine Learning-based web application that predicts student placement chances and provides personalized improvement guidance based on company requirements.

---

## 📌 Project Overview

This project predicts whether a student is likely to get placed using academic and skill-based features.
It also analyzes gaps between the student profile and company expectations, and generates:

* 📊 Placement probability
* 📈 Skill comparison graph
* ⚠️ Missing skills detection
* 💡 Personalized improvement suggestions
* 📅 4-week roadmap
* 🔗 Learning resources & networking links

---

## 🎯 Objectives

* Predict placement using Machine Learning
* Compare student profile with company expectations
* Provide actionable improvement plan
* Guide students with resources and roadmap

---

## 🧠 Technologies Used

* Python
* Streamlit (UI)
* Pandas, NumPy (Data Processing)
* Scikit-learn (ML Model)

---

## 📂 Dataset

Dataset includes:

* CGPA
* Internships
* Projects
* Workshops
* Aptitude Score
* Soft Skills
* Academic Marks (SSC, HSC)
* Placement Status

---

## ⚙️ Features

### 1. Prediction System

* Uses Logistic Regression
* Outputs probability of placement

### 2. Company-Based Analysis

* User selects target company
* System compares profile with company rubric

### 3. Visualization

* 📈 Line graph (Student vs Company)

### 4. Gap Analysis

* Shows strengths and missing skills

### 5. Improvement System

* Suggests improvements with links:

  * Aptitude practice
  * Communication skills
  * Projects
  * Internships

### 6. Weekly Roadmap

* 4-week structured plan for improvement

### 7. Networking Support

* LinkedIn connection suggestions

### 8. Download Report

* Includes:

  * Improvements
  * Resource links
  * Estimated time

---

## 📊 Machine Learning Model

* Algorithm: Logistic Regression
* Preprocessing: StandardScaler
* Evaluation:

  * Accuracy
  * ROC-AUC
